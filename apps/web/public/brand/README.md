# Brand assets

Replace with your final assets:

- **logo.png** — primary logo (e.g. on navy) — used for landing and sidebar when present
- **logo-transparent.png** — transparent background variant
- **favicon.ico** — browser tab icon (or use app/icon.png in Next.js)

The app falls back to `logo.svg` / `logo-transparent.svg` in this folder if PNGs are not present.
