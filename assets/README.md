# Brand assets

Place MY OWN AI MODEL logo files here. For use in the web app, copy into `apps/web/public/`:

- `logo.png` — primary logo (e.g. on navy)
- `logo-transparent.png` — transparent background

The landing and tenant sidebar use `apps/web/public/logo.svg` by default (text-only placeholder). Replace with your PNG assets and update references in `app/page.tsx` and tenant layout if desired.
